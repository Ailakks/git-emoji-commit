# simple-git-emoji

Simple CLI to encourage more concise commits.

## setup

* $ `npm i -g simple-git-emoji`
* $ `gc --help` to see options

## usage

* $`git add .` or create alias $`ga`
* $`gc --[option] "YOUR COMMIT MESSAGE"`
* $`git pull` or create alias $`gl`
* $`git push` or create alias $`gp`
